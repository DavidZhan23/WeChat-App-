# 体验权限配置指南

## 问题说明

当其他手机扫码预览小程序时，显示"暂无体验权限"，这是因为：
- 微信小程序默认只有开发者本人可以预览
- 其他用户需要被添加为"体验成员"才能扫码预览

## 解决方案

### 方案一：添加体验成员（推荐，适合测试）

#### 步骤1：登录微信公众平台

1. 访问：https://mp.weixin.qq.com
2. 使用小程序管理员账号登录

#### 步骤2：进入成员管理

1. 点击左侧菜单 **"成员管理"**
2. 或者：**"设置"** → **"成员管理"**

#### 步骤3：添加体验成员

1. 找到 **"体验成员"** 标签页
2. 点击 **"添加体验成员"** 按钮
3. 输入要添加的**微信账号**（微信号或绑定的手机号）
4. 点击 **"邀请"**

#### 步骤4：被添加成员确认

- 被添加的用户会收到微信通知
- 需要在微信中确认接受邀请
- 确认后就可以扫码预览了

### 方案二：上传体验版（适合多人测试）

#### 步骤1：上传代码

1. 在微信开发者工具中
2. 点击 **"上传"** 按钮
3. 填写版本号和项目备注
4. 点击 **"上传"**

#### 步骤2：设置为体验版

1. 登录微信公众平台
2. 进入 **"版本管理"**
3. 找到刚才上传的版本
4. 点击 **"选为体验版"**

#### 步骤3：配置体验版权限

1. 在 **"版本管理"** → **"体验版"** 中
2. 点击 **"体验成员"** 或 **"添加体验成员"**
3. 添加需要测试的用户

#### 步骤4：获取体验版二维码

1. 在 **"版本管理"** → **"体验版"** 中
2. 找到 **"体验版二维码"**
3. 下载或分享给测试用户
4. 用户扫码即可体验

### 方案三：使用真机调试（开发阶段推荐）

#### 步骤1：开启真机调试

1. 在微信开发者工具中
2. 点击 **"预览"** 或 **"真机调试"**
3. 生成二维码

#### 步骤2：扫码调试

1. 用**开发者本人的手机**扫码
2. 选择 **"真机调试"** 模式
3. 可以实时查看日志和调试

**注意**：真机调试只有开发者本人可以使用，其他用户无法使用。

## 详细操作步骤（方案一：添加体验成员）

### 1. 登录微信公众平台

访问：https://mp.weixin.qq.com

### 2. 找到成员管理

**路径1**：
- 左侧菜单 → **"成员管理"**

**路径2**：
- **"设置"** → **"成员管理"**

### 3. 添加体验成员

1. 点击 **"体验成员"** 标签
2. 点击 **"添加体验成员"** 或 **"+"** 按钮
3. 输入对方的**微信号**或**手机号**（需要对方已绑定微信）
4. 点击 **"邀请"** 或 **"确定"**

### 4. 对方确认

- 对方会收到微信服务通知
- 点击通知，确认接受邀请
- 确认后即可扫码预览

## 详细操作步骤（方案二：体验版）

### 1. 上传代码

在微信开发者工具中：
1. 点击顶部 **"上传"** 按钮
2. 填写：
   - **版本号**：例如 `1.0.0`
   - **项目备注**：例如 `测试版本`
3. 点击 **"上传"**

### 2. 设置为体验版

在微信公众平台：
1. 进入 **"版本管理"**
2. 找到 **"开发版本"** 标签
3. 找到刚才上传的版本
4. 点击 **"选为体验版"**

### 3. 添加体验成员

1. 进入 **"版本管理"** → **"体验版"**
2. 找到 **"体验成员"** 部分
3. 点击 **"添加"** 或 **"管理"**
4. 添加需要测试的用户

### 4. 分享体验版二维码

1. 在 **"体验版"** 页面
2. 找到 **"体验版二维码"**
3. 可以：
   - **下载二维码**：保存图片分享
   - **复制链接**：分享链接
   - **分享给朋友**：直接微信分享

## 三种方案对比

| 方案 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| **添加体验成员** | 少量用户测试 | 简单快速，实时预览 | 需要逐个添加 |
| **体验版** | 多人测试 | 稳定，可分享二维码 | 需要上传代码 |
| **真机调试** | 开发者本人 | 实时调试，查看日志 | 只能开发者使用 |

## 推荐方案

### 开发阶段
- **开发者本人**：使用 **"真机调试"**
- **少量测试用户**：使用 **"添加体验成员"**

### 测试阶段
- **多人测试**：使用 **"体验版"**
- **正式发布**：提交审核，发布正式版

## 常见问题

### Q1: 找不到"成员管理"菜单

**可能原因**：
- 账号权限不足（需要管理员权限）
- 菜单被隐藏

**解决**：
- 使用管理员账号登录
- 检查账号权限设置

### Q2: 添加体验成员时提示"用户不存在"

**原因**：
- 输入的微信号或手机号不正确
- 对方没有绑定微信

**解决**：
- 确认对方的微信号
- 让对方先绑定微信

### Q3: 体验成员扫码还是显示"暂无体验权限"

**检查**：
1. 对方是否已确认接受邀请
2. 是否使用的是预览二维码（不是体验版二维码）
3. 账号是否正确

### Q4: 体验版和预览版的区别

- **预览版**：开发者工具生成的二维码，需要添加体验成员
- **体验版**：上传代码后设置的版本，更稳定，可分享

3. **添加体验成员**
4. 分享**体验版二维码**
